### Option 2: Vercel (Also Very Easy)

**Steps:**
1. Go to [https://vercel.com](https://vercel.com)
2. Sign up for free account
3. Click "Add New Project"
4. Drag and drop your `LeafyPop` folder
5. Click "Deploy"
6. Your site will be live in seconds!

---

### Option 3: GitHub Pages (Free)

**Steps:**
1. Create a GitHub account at [https://github.com](https://github.com)
2. Create a new repository (e.g., "leafypop-website")
3. Upload all your files from the `LeafyPop` folder to the repository
4. Go to Settings â†’ Pages
5. Select the main branch and `/root` folder
6. Your site will be available at: `your-username.github.io/leafypop-website`

---

### Option 4: Cloudflare Pages (Free & Fast)

**Steps:**
1. Go to [https://pages.cloudflare.com](https://pages.cloudflare.com)
2. Sign up for free account
3. Click "Create a project"
4. Upload your `LeafyPop` folder
5. Deploy!
